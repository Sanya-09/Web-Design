HTML (HyperText Markup Language)

HTML is the standard markup language for creating web pages. It describes the structure of a web page.

<html> <head> <title>Page Title</title> </head> <body> <h1>Heading</h1> <p>Paragraph</p> </body> </html>

Elements are the basic building blocks of HTML pages. Elements are represented by tags.

  • <p> for paragraphs
  • <h1> to <h6> for headings
  • <img> for images

...

CSS (Cascading Style Sheets)

CSS is used to style HTML elements. It controls the layout, colors, fonts, and other visual aspects of a webpage.

body { background-color: lightblue; }

CSS rules are made of selectors and declarations.

  • Selectors target HTML elements.
  • Declarations define the styles.
JavaScript

JavaScript is a scripting language that makes web pages interactive.

console.log("Hello, world!");

It can manipulate the DOM, handle events, and make asynchronous requests.

  • Variables
  • Functions
Git

Git is a distributed version control system for tracking changes in source code during software development.

git commit -m "Initial commit"

It is designed for coordinating work among programmers, but it can be used to track changes in any set of files.

  • Commit
  • Branch
Deployment

Deployment is the process of releasing software to users.

npm run build
